# V2bX
A V2board node server based on Sing-Box.

一个基于Sing-Box的V2board节点服务端，支持Vmess,Vless,Trojan,TUIC,Shadowsocks,AnyTLS,Hysteria,Naive协议

Find the source code here: [ouyangmland/V2bX-singbox](https://github.com/ouyangmland/V2bX-singbox)

# 一键安装

```
wget -N https://raw.githubusercontent.com/ouyangmland/V2bX-script/master/install.sh && bash install.sh
```
